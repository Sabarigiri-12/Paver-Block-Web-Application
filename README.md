This project is a responsive web application developed to showcase different types of paver block designs and allow users to select products and enter project-related details.

 Features
      Display of various paver block designs
      User-friendly and responsive UI
      Option to select block types
      Input fields for project details (area, price, location, etc.)
      Clean and simple interface

Technologies Used:
         HTML
         CSS
         JavaScript
         React.js 
         MySQL 

Purpose:
    The application is designed to help customers easily explore paver block options and provide necessary details for project estimation.
